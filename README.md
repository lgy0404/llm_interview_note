## 简介

本仓库为大模型面试相关概念，由本人参考网络资源整理，欢迎阅读，如果对你有用，麻烦点一下 `start`，谢谢！

为了在低资源情况下，学习大模型，进行动手实践，创建 [tiny-llama2-zh](https://github.com/wdndev/tiny-llama2-zh)仓库，旨在构建一个小参数量的中文Llama2大语言模型，方便学习，欢迎学习交流。
